# David Okonkwo — PLS Review Checklist (Tunnel / Terrestrial LiDAR)

## Pre-Review Verification
- [ ] Task specification document received and reviewed
- [ ] Operator credentials verified (confined space cert, insurance)
- [ ] Confined space entry logs and atmospheric monitoring records received
- [ ] GC terms reviewed for accuracy requirements

## Scan Registration
- [ ] Registration error (RMS) within specification (typically <=0.02 ft)
- [ ] Sufficient overlap between scan positions
- [ ] Target-based or cloud-to-cloud registration method documented
- [ ] Control points tied to surface survey network
- [ ] No registration drift along tunnel length

## Point Spacing and Coverage
- [ ] Point spacing meets specification (<=0.05 ft on structural elements)
- [ ] Full 360-degree coverage at each scan position
- [ ] No shadow zones behind obstructions (or documented)
- [ ] Coverage continuous — no gaps between scan positions
- [ ] Lateral connections and junction chambers fully captured

## Tunnel Geometry
- [ ] Cross-sections extracted at specified interval (25-ft typical)
- [ ] Cross-section shape accurately represents as-built condition
- [ ] Invert profile extracted and consistent with design grade
- [ ] Diameter/dimension measurements at key locations
- [ ] Alignment (horizontal and vertical) computed and documented

## Wall Thickness Measurements
- [ ] Measurement locations documented with station and offset
- [ ] Measurement method documented (direct, GPR, ultrasonic)
- [ ] Minimum/maximum thickness identified
- [ ] Areas of reduced thickness flagged for structural review
- [ ] Comparison to design thickness where available

## Structural Condition Documentation
- [ ] Spalling locations mapped with extent and depth
- [ ] Crack patterns documented (location, orientation, width)
- [ ] Joint condition assessed (separation, offset, sealant condition)
- [ ] Corrosion/deterioration areas mapped
- [ ] Root intrusion locations documented
- [ ] Sediment depth recorded at specified intervals

## Deliverable Completeness
- [ ] e57 registered point cloud — correct format, metadata
- [ ] LAS classified point cloud — consistent classification
- [ ] DXF cross-sections — correct interval, labeling, scale
- [ ] Tunnel condition report — complete, photo-documented
- [ ] Invert profile — LandXML format, correct datum
- [ ] Wall thickness log — CSV or tabular format
- [ ] Confined space entry logs — complete, signed

## Coordinate System
- [ ] Tied to surface control network (same datum as corridor survey)
- [ ] Michigan State Plane South, NAD83(2011), NAVD88
- [ ] Transformation between tunnel and surface coordinates documented
- [ ] Units confirmed (US Survey Feet)

## Safety Compliance Review
- [ ] Confined space permits complete and on file
- [ ] Atmospheric monitoring logs show safe conditions throughout
- [ ] Rescue plan was in place during all entries
- [ ] No safety incidents reported

## Final Certification
- [ ] All checklist items pass or documented exceptions noted
- [ ] Digital PLS seal applied per MCL 339.2007
- [ ] Certification date and license number on all deliverable sheets
- [ ] Signed certification letter generated
- [ ] Structural concerns flagged for engineer review (if any)
