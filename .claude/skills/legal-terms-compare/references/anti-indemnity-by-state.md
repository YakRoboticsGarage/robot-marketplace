# Anti-Indemnity Statutes by State

Source: https://www.mwl-law.com/wp-content/uploads/2013/03/Anti-Indemnity-Statutes-In-All-50-States-00131938.pdf

## Michigan (MCL 691.991)

**Applies to:** Public works projects
**Prohibits:** Broad-form indemnity — cannot require a party to indemnify for damages caused by the sole negligence of the indemnitee
**Allows:** Intermediate indemnity (each party responsible for own proportional fault)
**Key detail:** Contracts with public entities must not violate comparative fault principles. Licensed architects, engineers, and surveyors cannot be required to defend public entities from negligence claims.

## Texas (Tex. Ins. Code § 151.102)

**Applies to:** All construction contracts
**Prohibits:** Both broad-form and intermediate indemnity in construction contracts (strongest anti-indemnity state)
**Allows:** Only comparative fault indemnity (each party indemnifies only for own negligence percentage)

## California (Cal. Civ. Code § 2782)

**Applies to:** All construction contracts (residential and commercial)
**Prohibits:** Broad-form indemnity
**Allows:** Intermediate indemnity (type I)
**Key detail:** Separate rules for residential vs commercial

## Ohio (ORC § 2305.31)

**Applies to:** All construction contracts
**Prohibits:** Broad-form indemnity
**Allows:** Intermediate indemnity

## Florida (Fla. Stat. § 725.06)

**Applies to:** All construction contracts
**Prohibits:** Broad-form indemnity
**Allows:** Intermediate indemnity up to $1M or contract value (whichever less) with specific insurance requirements

## States with NO anti-indemnity statute

Several states have no anti-indemnity statute for construction: AL, DC, KY, ME, MO, NH, PA, VT, WI, WY. In these states, even broad-form indemnity may be enforceable.

## How the Skill Uses This

When comparing contract terms:
1. Identify the project state
2. Look up the applicable statute
3. If the subcontract contains broad-form indemnity AND the state prohibits it → flag as UNENFORCEABLE
4. If intermediate but the state only allows comparative → flag as UNENFORCEABLE
5. Include the statute citation in the output
